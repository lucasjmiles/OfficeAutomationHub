# Basic .NET WPF App (XAML)

This project is a minimal WPF application built with .NET and XAML.

## What it does

- Uses **WPF** for the desktop UI.
- Defines the UI in **XAML** (views) and connects behavior via code-behind.

## Getting started

### Prerequisites

- .NET SDK (version matching the project’s target framework)

### Run

```bash
dotnet run
```
